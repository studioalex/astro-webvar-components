The Custom Button Component is a lightweight, customizable button solution built with HTML, Tailwind CSS, and JavaScript, without relying on any frameworks. It's designed to minimize footprint and integration issues, making it an ideal choice for projects where performance and flexibility are paramount.

## Features

-  **Variants**: The button supports a wide range of color variants, including **primary**, **secondary**, **success**, **danger**, **warning**, **info**, **light**, and **dark**. Each variant comes in two styles: *solid* and **outline**, providing a versatile look for different use cases.
-  **Rounded Corners**: The component allows for customization of the button's rounded corners, with options ranging from none to fully rounded. This feature ensures that the button can fit seamlessly into any design aesthetic.
-  **Sizes**: The button is available in various sizes, from extra small (xs) to extra large (xl), allowing developers to choose the perfect size for their application's needs.
-  **Custom Classes**: Developers can extend the button's functionality and appearance by adding custom CSS classes through the `class` property.
-  **Accessibility and Interactivity**: The button is designed with accessibility in mind, including focus and disabled states. It also features hover and focus effects for enhanced user interaction.


## Usage

To use the Custom Button Component, simply include it in your HTML and customize it using the available properties. The component is built with Astro, a front-end framework that allows for the use of TypeScript and JSX, making it easy to integrate into modern web projects.

## Props

| Property          | Type     | Default     | Description                                                                                      |
|-------------------|----------|-------------|--------------------------------------------------------------------------------------------------|
| Variants          | string   | 'secondary' | Determines the color variant of the link button. Options include:                                |
|                   |          |             | `primary` / `primary-outline`                                                                    |
|                   |          |             | `secondary` / `secondary-outline`                                                                |
|                   |          |             | `success` / `success-outline`                                                                    |
|                   |          |             | `danger` / `danger-outline`                                                                      |
|                   |          |             | `warning` / `warning-outline`                                                                    |
|                   |          |             | `info` / `info-outline`                                                                          |
|                   |          |             | `light` / `light-outline`                                                                        |
|                   |          |             | `dark` / `dark-outline`                                                                          |
| Rounded Corners   | string   | 'full'      | Customizes the rounded corners of the Link Button. Options include:                              |
|                   |          |             | `none`, `full`, `sm`, `md`, `xl`                                                                 |
| Sizes             | string   | 'md'        | Sets the size of the Link Button. Options include:                                               |
|                   |          |             | `xs`, `sm`, `md`, `lg`, `xl`                                                                     |
| Additional HTML Button Properties | - | - | All properties of a standard [HTML button](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button) element are also accepted by this component. This includes attributes such as `disabled`, `form`, `formaction`, `formenctype`, `formmethod`, `formnovalidate`, `formtarget`, `name`, `type`, `value`, and event handlers like `onclick`, `onfocus`, etc. |
